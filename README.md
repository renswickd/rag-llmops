# End-to-End RAG with LLMOps
This project demonstrates a production-grade Retrieval-Augmented Generation (RAG) system integrated with LLMOps practices to ensure reliability, scalability, and continuous improvement.

# 🚀 Key Features
- Automated Ingestion Pipeline: Scalable document processing, semantic chunking, and automated embedding updates.
- Advanced Retrieval: Hybrid search (vector + keyword) with re-ranking to improve context relevance.
- LLMOps CI/CD: Automated testing for prompt regressions and RAG evaluation metrics using GitHub Actions.
- Observability & Tracing: Full-stack tracing of every query, including retrieval latency and token costs.
- Evaluation Framework: Systematic scoring of Faithfulness, Answer Relevance, and Context Precision using [RAGAS/DeepEval].
